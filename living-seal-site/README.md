# Living Seal site

A two-version React/Vite site based on the supplied Living Seal brand direction.

- Original cinematic site: `/`
- Minimal image-led site: `/minimal.html`

## Run locally

```sh
npm install
npm run dev
```

Both versions are built together by `npm run build` and keep separate React and CSS entry points.

The private AI prompt is an in-browser interaction demo and does not send data to a server.
